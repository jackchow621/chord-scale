1、安装mingus
   https://github.com/edudobay/python-mingus.git
   如果是python3，需先切换到python3分支,再进行安装
   git checkout python3 
   python setup.py install

2、下载qsynth-0.3.6-setup.exe 并安装，注意安装路径。
    https://sourceforge.net/projects/qsynth/files/qsynth%20%28attic%29/
    安装完成后，配置环境变量 fluidsynth 指向上述安装路径，并配置到PATH变量中 %PATH%;%fluidsynth%

3、pip install PyQt5
4、pip install BeautifulSoup4


可选：
1、安装pygame：pip install pygame
2、下载sf2格式的声源，放到当前路径。
    本例用的是钢琴音源：grand-piano-YDP-20160804.sf2
    下载路径：https://musescore.org/en/handbook/soundfonts-and-sfz-files#install-sfz
    搜索关键字：SF2 Pianos  或 Acoustic grand piano, release 2016-08-04
